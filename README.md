## 使用第三方库

+ OpenGL

+ glad

## 注意事项

+ 这个分支已经交付

+ 本模块原理远没有大家想象中的复杂，但是其中包含非常多的面向对象设计模式，非常欢迎大家阅读修改。

## 已经实现

+ 定点着色器和片段着色器加载
+ 立方体和球体的创建
+ 键盘和鼠标的总体控制
+ 基础phone光照渲染
+ obj文件导入
+ 纹理贴图支持
+ 实例化渲染
+ cube 等集合体的拓展，强制硬编码可以显著减少游戏加载时常（包括多面上色纹理，
+ camera类封装
+ 多光源
