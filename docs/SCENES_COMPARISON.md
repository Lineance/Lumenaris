## 🌀 场景 1: 螺旋塔 (Spiral Tower)

### 视觉特点
- **高耸的塔形结构**，向上延伸
- **螺旋上升**，每层旋转 0.5 弧度
- **底部大顶部小**（尺寸从 1.0 到 0.5）

### 颜色
- **火焰渐变**：底部红色 → 顶部黄色
- RGB 变化：
  - R: 固定 1.0
  - G: 0.3 → 1.0
  - B: 0.4 → 0.1

### 参数
```cpp
层数: 40
每层立方体: 8
层高: 1.0
螺旋半径: 6.0
总高度: 40 单位
```

### 摄像机位置
```
(0, 25, 30) - 从稍高的位置仰视塔
```

### 视觉效果
✅ **最容易识别**：
- 高耸的塔形结构
- 明显的螺旋纹理
- 像火焰般的颜色

---

## 🌊 场景 2: 波浪地板 (Wave Floor)

### 视觉特点
- **平坦的地板**，铺满整个视野
- **压扁的立方体**（Y 轴缩放 0.3），形成平板效果
- **复杂的波浪**，两个正弦波叠加
- **无旋转**，所有立方体面向一致

### 颜色
- **海洋蓝绿色**：中心蓝绿 → 边缘深蓝
- RGB 变化：
  - R: 固定 0.0
  - G: 0.3 → 0.7
  - B: 1.0 → 0.6

### 参数
```cpp
网格大小: 25 × 25
间距: 1.8
波浪高度: ±3.5
立方体形状: 1.0 × 0.3 × 1.0 (压扁)
```

### 摄像机位置
```
(0, 15, 50) - 俯视波浪地板
```

### 视觉效果
✅ **与场景 1 完全不同**：
- 水平分布（不是垂直）
- 压扁的立方体（不是完整立方体）
- 蓝绿色系（不是红黄色系）
- 像海面一样的波浪

---

## 🏝️ 场景 3: 浮动群岛 (Floating Islands)

### 视觉特点
- **12 个浮动的岛屿**，随机分布在空间中
- **球形簇**，每个岛屿是一个压扁的球体
- **完全随机旋转**，每个立方体方向不同
- **不同高度的岛屿**，从 Y=8 到 Y=20

### 颜色
- **紫色/粉色/橙色系**：每个岛屿独特
- RGB 范围：
  - R: 0.6 ~ 1.0
  - G: 0.0 ~ 0.6
  - B: 0.1 ~ 0.9

### 参数
```cpp
岛屿数量: 12
每岛立方体: 100
分布范围: ±25 (XZ平面)
高度范围: 8 ~ 20
随机种子: 42 (固定，每次运行相同)
```

### 摄像机位置
```
(0, 20, 55) - 观察群岛分布
```

### 视觉效果
✅ **最独特的场景**：
- 3D 空间随机分布（不是规则排列）
- 球形岛屿（不是平面或螺旋）
- 随机旋转（混乱但有序）
- 像外星世界的浮岛

## 🎯 关键差异总结

### 形状差异
1. **场景 1**: 垂直塔 → 🗼
2. **场景 2**: 水平地板 → 🌊
3. **场景 3**: 3D 浮岛 → 🏝️

### 颜色差异
1. **场景 1**: 暖色调（红→黄）🔥
2. **场景 2**: 冷色调（蓝绿）🌊
3. **场景 3**: 彩色调（紫粉橙）🌈

### 几何差异
1. **场景 1**: 完整立方体，有旋转
2. **场景 2**: **压扁立方体，无旋转** ⚡
3. **场景 3**: 完整立方体，随机旋转

### 排列差异
1. **场景 1**: 规则螺旋
2. **场景 2**: 规则网格
3. **场景 3**: 随机球形

## 🔍 如何区分场景

### 最明显的特征
1. **场景 1**: 抬头看，看到高塔
2. **场景 2**: 俯视，看到波浪地板
3. **场景 3**: 四处看，看到浮岛

### 颜色快速识别
1. **红色** → 场景 1
2. **蓝色** → 场景 2
3. **紫色/粉色** → 场景 3

### 立方体形状
1. **完整立方体** + 旋转 → 场景 1 或 3
2. **压扁立方体** + 不旋转 → 场景 2
